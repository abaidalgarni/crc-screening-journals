# CRC Screening Barriers Study — Journal Recommendations

Interactive single-page report ranking **30 MEDLINE-indexed journals** for a cross-sectional KAP study on colorectal cancer screening barriers.

## Features

- 30 journals ranked by acceptance likelihood across 3 tiers
- All journals confirmed MEDLINE-indexed and active as of Feb 2026
- Sortable/searchable table with tier filtering
- Recommended submission cascade strategy
- Dark mode toggle
- Mobile responsive design
- Print-friendly layout

## Exclusions

Cureus, APJCP, PLOS ONE, all Saudi journals, BMJ Open, IJERPH, Heliyon, Baishideng journals
